制作の経緯

勤務先の会社が運営する中古品買取店の集客アップを見込んだ集客用LPリニューアルPJにて、コーディングを担当しました。

使用技術

  • Visual Studio Code
  • WordPress
  • Google Maps Platform

アピールポイント

URLパラメーターで指定したエリアによってページ内容の出し分けを実装

URLパラメーターで指定したエリアにより、ページ内に表示する店舗などが切り替わるように実装しました。
URLパラメーターで指定したエリアと一致するWordPressのカスタムタクソノミーに紐づく店舗情報（カスタム投稿で登録）を呼び出すように実装しました。

「現在地から近くの店舗を検索」ボタンを押下することで、近くの店舗を現在地から近い順に表示する機能を実装

GPSで取得したユーザーの位置情報と、そのエリアを対応する店舗の位置情報の距離をGoogle Maps PlatformのAPIで計測し、店舗情報の一覧を距離の近い順に表示する機能を実装しました。

買取実績のカルーセルをSwiperで実装

カスタム投稿で登録した買取実績を、URLパラメーターで指定したエリアで呼び出して表示するように実装しています。
ウィンドウ幅に応じてスライドの大きさや数をレスポンシブに調整するのに意外と苦労しました。
